字根一：~duc = ~duce = ~duct = lead = 引导
===

```
1. seduce = se~ (away) + duce (lead) = 引导(某人)到别处去
                                     = (vt.)使误入歧途
   ★注意：seduce 一定是「引导别人去做坏事」，allure 是「放饵诱惑别人
           上钩」，enchant 是「说甜言蜜语使人迷醉」，tempt 是「为了
           某严苛的目的而试探、试炼」~~ 这些字以后都会带到，先做字辨。

2. induce = in~ (in) + duce (lead) = 引到里头来 = (vt.)引发、导致
                                                = cause = bring forth

3. deduce = de~ (down) + duce (lead) = 把(事物)一直往下引导 = (vt.)推理

4. conduce = con~ (together) + duce (lead) = 把(事物)都引导至某一方向
                                           = (vi.)贡献
                                           = contribute

5. conduct = con~ (together) + duct (lead) = 大家共同引导的事物
                                           = (n.)行为举止 = behavior
                                           = 引导大家的行为
                                           = (vt.)引导、指挥 = direct

6. reduce = re~ (back) + duce (lead) = 把(事物)向后引导 = (v.)减退、降低

7. produce = pro~ (forward) + duce (lead) = 将事物向前引导
                                          = (vt.)产生、制造

8. adduce = ad~ (去使) + duce (lead) = 去把事物引导来此
                                     = (vt.)举证、提供证明

9. introduce = intro~ (相互之间) + duce (lead) = 把人或事物引导至人们之间
                                     = (vt.)引入、介绍

10. subdue = sub~ (向下) + due (duce的变体，lead) = 把某人带到自己下头去
                                     = (vt.)使屈服、臣服

11. educate = e~ (ex~ 的变体，向外) + duc (lead) + ~ate (使得)
                                          = 把(某人)带领出来
                                          = (vt.)教育、教导

  ★注意：educate 既然是把迷途羔羊引导上正途，必然是长期而正式的
          project，train 是为了某特殊目的而加以训练，school 是使
          人领会到艰难的事物，常指短期或非正式学习途径，tutor 则是
          私下教导了，先做字辨。

12. ductile = duct (lead) + ~ile (~ible的变体，能够的)
                                          = 能够导引的
                                          = (adj.)可排放出去的

13. duct = lead = (n.)导管


14. duke = duc 的变体 = 领导 = (n.)公爵
                             --> duchess (n.)女公爵

15. viaduct = via~ (道路) + duct (引导) = 引导用道路
                                        = (n.)山谷间长途穿越高架道路

16. abduct = ab~ (from) + duct (引导) = 从某地把某人引领走
                                      = (vt.)绑架
                                      = kidnap
```
