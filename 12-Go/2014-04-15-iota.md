
iota
=========

关键字iota定义常量组中从0开始按行计数的自增枚举值。
```go
const (
    Sunday = iota       // 0
    Monday              // 1，通常省略后续行表达式
    Tuesday             // 2
    Wednesday           // 3
    Thursday            // 4
    Friday              // 5
    Saturday            // 6
)

const (
    _ = iota                    // iota = 0
    KB int64 = 1 << (10*iota)   // iota = 1
    MB                          // 与KB表达式相同，但 iota = 2
    GB
    TB
)
```
- 在同一常量组中，可以提供多个iota，它们各自增长。

```go
const (
    A,B = iota,iota << 10   // 0, 0 << 10
    C, D                    // 1, 1 << 10
)
```
- 如果iota自增被打断，须显式恢复。

```go
const (
    A = iota            // 0
    B                   // 1
    C = "c"             // c
    D                   // c，与上一行相同
    E = iota            // 4，显式恢复。注意计数包含了C、D两行
    F                   // 5
)
```
