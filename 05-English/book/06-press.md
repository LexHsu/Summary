字根五：-press = 压
===

```
1. press = 压的动作
         = (vt.)压挤、敦促、紧迫
         = (n.)印刷、新闻界

         当「新闻界」解是很新的含意，但是很明显是从「印刷」演变成「新闻报导」，进而衍生为「新闻业者」的。

2. pressure = press (压) + -ure (动作状态)
            = 压下的状态
            = (n.)压力、压迫感

            这个字有一个新创字pressurize 是个新闻常用字，叫做「对...施压」
            (= put pressure on)，字尾-ize 本来就是「使得」的动词字尾。

3. compress = com- (一起) + press (压)
            = 把事物都压在一起
            = (vt.)压缩、浓缩

4. depress = de- (向下) + press (压)
           = 把(某人)往下压
           = (vt.)使灰心沮丧
           = disappoint
           = deject
           = let down

5. express = ex- (向外) + press (压)
           = 把(事物)压送出去
           = (vt.)表达、快递寄出
           = (n.)特快车 (adj.)快速的

           express 的多种含意都跟「向外压送出」相关，很好理解，
           expression 叫「用语」也叫「表情」
           因为表达只有两种，有声表达出的事物叫做「用语、措辞」，
           无声表达出的事物叫做「表情」

6. impress = im- (往内) + press (压)
           = 把(人事物)往内压进心中
           = (vt.)使留下印象

           impression 有个形容词叫做impressionable，看似很难，其实容易：
           -able 是「能够的」的意思，「能够轻易就使他有印象」，英文就叫做「他容易受影响」

7. repress = re- (向后) + press (压)
           = 把(事物)往后压回去
           = (vt.)抑制、强忍
           = hold back

           这个字不好用，常用在负面的情况下，比如说 a repressed boy 是指家教过严，以致于孩子被压抑过当。

8. oppress = op- (ob- 的变体，朝着) + press (压)
           = 朝着(事物)压下去
           = (vt.)压迫

9. suppress = sup- (sub- 的变体，向下) + press (压)
            = 把(人事物)向下压
            = (vt.)镇压、查禁

            通过字根，很容易辨别 repress、oppress、和suppress 的区别：
            repress 一定是用于「压抑」情感，
            oppress 只是指「被压迫」：the oppressed farmers 是指「这些农夫遭受暴政被压迫」，并没有被武力镇压。
            suppress 一定是指有了暴乱，因此才以武力或法令去镇压，如：to suppress the rebellious farmers。
            这就是为什么oppressor ​​叫做「暴君」，而suppressor ​​叫「镇压者」。
```
