Linux 终端命令
===

### 管道命令

通过管道命令可把一个命令的输出结果转移给另一个命令。如 `ls` 命令会显示当前目录中所有的文件列表，而`grep` 命令会在制定的地方搜索输入的字符串。把这两者通过管道命令 `|` 结合起来，即可在当前目录搜索某个文件，如： `ls | grep really`。

### 通配符

星号 `*` 是一个可以匹配任何东西的通配符。如把 "really long file name" 和 "really very long file name" 从当前目录都删除：`rm really*name`。

### 输出重定向

`>` 可把一个命令的输出结果重定向到一个文件，而不需要额外使用另一个命令。如用 `ls` 命令列出当前目录的所有文件，并把输出的列表输入到了一个名为`file`，而不是仅仅输出显示在终端上。`ls < file`。

### 命令行历史

Bash记住使用过的历史命令。你可以使用向上键和向下键来翻阅你已经使用过的命令。使用”history”命令会把历史命令都打印出来，因此你可以使用管道命令来搜索你最近使用的命令。

### 后台运行命令

`&` 符号使 Bash 在后台运行该程序：如 `firefox &`。

### 连接命令
`&&`支持第一个命令成功运行完毕后接着运行第二个命令，如 `sleep` 命令会以秒为单位接受一个参数，然后倒计时，允许结束。单独来用的话，该命令没有任何用处，但是可讲它作为运行下一个程序之前的延时。下面的命令会停5秒然后运行gnome的截屏工具：`sleep 5 && gnome-screenshot`。
