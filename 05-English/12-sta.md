字根十一】~sta = ~stat = ~stitute = 站
===

```
1. stand = sta (站) + ~and (~ant的变体，本质相关)
         = 站的动作
         = (vi.)站 (vt.)承受 (n.)摊位、立场

         stand 当「承受」解时，是凸显面对困难始终「屹立」的那个意象，
         而名词做「摊位」解也并不意外。字尾由 ~ant 变成 ~and 是受到 standard 的连带改变。


2. standard = sta (站) + ~and (本质相关) + ~ard (人或物)
            = 竖立在那里的事物
            = (n.)标准、支柱

            ~ard 其实并不是罕见的字尾，因为 drunkard、placard、coward、bastard、leopard...等这些字都是常见字，
            常指「比较劣等」的事物，跟 ~ster 这个字尾意义差不多（如 monster、administrate）。
            把比较不好的事物摆在那里，原本意义叫做「警诫」，是个要超越的对象，因此演变为「标准」。
            有人误把这个字跟 ideal 混用，但是 ideal 是用来「达到」的，而 standard 是用来「超越」和「改变」的。
            standard 要比stantard 好发音，因此用 ~d~ 而不用 ~t~。


3. stable = sta (站) + ~able (能够...的)
          = 能够一直站立的
          = (adj.)稳固的、踏实的

          这个字修饰人或事物皆可，女孩子常梦想找到一个 stable man who earns a stable income，
          它有个相反词叫做 unstable，但是很吊诡的是：stability 名词的相反词却是 instability。


4. state = sta (站) + ~ate (使得)
         = 使得(事物)​​屹立
         = (n.)情况、地位、政府 (vt.)陈述

         这个字原本是动词，叫做「使事物一直持续」，所以慢慢变成「开口陈述」，古时候的人没有笔，要使得事物流传只能靠嘴巴。
         而名词的意义反而越来越常用，逐渐取代动词的重要性了。 state 名词的意义并没有脱离字源太远，
         「人事物屹立的状态」叫做情况、地位，而一大堆有关系的人屹立的状态就变成政府了。
         从这个多义字衍生出 estate，叫做「房地产」：那个 e~ 是 ex~ 的的变化：「屹立在外头」让人看到的事物，叫做「房屋」，
         但是由于 state 是抽象的状态，所以 estate 也演变成抽象的「房地产」。
         我有一次在宴会上听到一个老外问我旁边的穿西装男士：「What do you do?」这位体面男士回答：「I sell houses.」
         我偷看到那老外迷惑的表情，真想捧腹大笑。 「卖房子」是很好赚的职业，但是你一讲 sell houses，老外只会以为你买房子住，
         又把一堆自己的房子卖出去，如果靠「卖（自己）房子」为职业，你听到不会吓到吗？真正的讲法是：I sell estate.。
         像这位体面男士他的职业应该就是estate agent，在美国也有人称为realtor。

5. station = sta (站) + ~tion (事物结果)
           = (事物)站立的结果
           = (n.)车站、局、台

           可想成是「很多人在那里站着」，所以叫做「车站」，不一定我的思考就是标准答案，
           古人造字这么久了，当初的造字逻辑今天早已不复得知了，只能靠想像力来弥补中间的差距。
           这个字的形容词 stationary 反而更接近字源，
           但是要跟stationery 区分，后者是「站在桌上」的那些事物，叫做「文具」。


6. stance = sta (站) + ~ance (本质相关物)
          = 站立的状态
          = (n.)姿态


7. statue = sta (站) + ~tue (~ture的变体，动作结果)
          = (事物)站立的结果
          = (n.)雕像

          ~sta 是个相当大的字根，衍生字数量太多
          statue 和 stature 是一模一样的字，只是后者意义演变为「身材」，
          而 statue 只好变个样子以资区别啰！


8. statute = sta (站) + ~tu (~tus的变体，拉丁阳性名词字尾) + ~te (~ate的变体，使得)
           = 使得(事物)​​站立的事物
           = (n.)成文法

           这个字超文言，来自于拉丁文 status，拉丁字尾 ~us 没有什么意义，只是表示阳性名词
           （中性名词通常是~um，而阴性通常是~a），这是status 再加上~ate 的简化结果，
           原意是古罗马的法令是写在木表或石碑上矗立在人们面前。造字过程很繁复，
           但是反正字义冷僻，跟其他相似字分辨的出来即可。不过status 这个字倒是常出现，这是「地位」的意思。

9. static = stat (站) + ~ic (相关性质的)
          = 与站着相关的
          = (adj.)静止的、不活跃的

          这个字有个很难的衍生字，叫做 statistics「统计学」，字尾~istic 其实跟~ic 是一模一样的意义，
          再加上~s 只是表示多种事物，因此变成「学科」。统计学是对于「站着的事物」（诸如房屋、公司...等）做研究的领域。

10. instant = in~ (在...之内) + sta (站) + ~ant (本质相关的)
            = (事情一发生)就站到里头来的
            = (adj.)立即的(n.)瞬间

11. distant = dis~ (离开) + sta (站) + ~ant (本质相关的)
            = 事物站离开的
            = (adj.)远离的、疏远的
            = remote

            两个事物站离开，当然就是「远离」，它的名词 distance 不是「远离」，而是「一段距离」，


12. extant = ex~ (在外) + sta (站) + ~ant (本质相关的)
           = (事物)还在外头站着
           = (adj.)遗留的、残存的
           = existing

           历经这么久时间，还在外头屹立的当然是「遗留下而残存的」东西。


13. constant = con~ (共同) + sta (站) + ~ant (本质相关的)
             = (事物)一起屹立着的状况
             = (adj.)固定不变的、持续的、忠贞的

             与 frequent 区别：
             constant 的字源来看就知道这个字比较偏向 always 的意义，「始终不变」跟「经常」并不完全相等，
             而且frequent 是断续的经常，而 constant 是始终的经常。


14. substance = sub~ (在...之下) + sta (站) + ~ance (本质相关状态)
              = 位于(某事)其下(的事物)
              = (n.)本质、基本内容、物质

              支撑于某物体其下的事物叫做「本质」。而这个字的形容词是 substantial。
              这个字文言，日常生活中常用 stuff 来表达即可。


15. circumstance = circum~ (四周、环绕) + sta (站) + ~ance (本质相关状态)
                 = 在(事物)四周站着的状态
                 = (n.)情况、境遇

                 circumstance 这个字看起来很难，其实很常用。它跟 situation 不同，
                 前者指某人四周的「生活状况」以及情势，范围较小；而situation 泛指「某地当时的状况」，
                 可能是社会、可能是国家，范围较大，所以「个人情况」可能两者皆可，
                 但是「国家现况」就只能用situation。circum~ 这个字首慢慢越来越常见。


16. obstacle = ob~ (against，朝向、面对) + sta (站) + ~acle (小器具)
             = 矗立在眼前的小物体
             = (n.)障碍物、绊脚石
             = block

             ob~ 这个字首用 against 来解比较周详，中文翻成「朝向」或「反对」都不适当，它只是「相对」的意思。
             而 ~acle 看似陌生，其实不然，spectacle 叫做「眼镜」，不就是「用来看的小器具」吗？
             把 ~acle想成 device 即可。


17. establish = e~ (ex~的变体，在外) + sta (站) + ~bl (~ble 的变体，能够...的) + ~ish (属于的)
              = 使得(某事物)能够站立在外头的= (vt.)建立、创立
              = set up
              = form
              = found

              因为字根已经有了 ~s~，所以 ex~ 简化为e~，有细心的读者会问：
              expect也是ex~ 加上 spect 啊，为什么不拼做 espect 呢？
              这是因为 espect 的发音跟 aspect 的发音一样，为了区分才变成 expect 的。


18. ecstasy = ec~ (ex~的变体，往外) + sta (站) + ~sy (~sis的变体，状态)
            = 站离开(到正常状况)外头的状态
            = (n.)狂喜、出神

            这个字变化的很奇特，因为在拉丁文中是extasis ~~ 彻底符合字源以及造字原则，
            虽然目前语源学家没有答案，我在想可能是受到一些希腊字，如 eccentric、ecclesiastic...
            误以为这个是同一类的字才造成拼音误植的结果。


19. stage = sta (站) + ~age (集合物)
          = 人集合在一起站着的事物
          = (n.)讲台、舞台、阶段 (vt.)举办

          ~age 其实也是蛮常见的，只是大家对于字尾一般比较不熟，所以容易忽略掉，
          诸如：village、marriage、bondage...等应该都不是陌生字，
          ~age 通常指「事物集合的状态」，所以 stage 这个字起源于戏剧，
          大家共同站在上头的状态，变成「舞台」，「阶段」这个意思也是由换舞台而演变而成的。


20. stall = sta (站) + ~all (~al的变体，相关的)
          = 让(人、动物)站着的地方
          = (n.)馬廄、货摊

          因为 stal 让人比较判别不出发音，所以才慢慢变成 stall 的。
          这个字最原始就是让马站着的地方，最近衍生出的意义是当动词「拖延」。


21. steady = stead (stat 的变体，站) + ~y (充满...的)
           = 一直持续站着的
           = (adj.)稳定的、持续的、稳重可靠的

           steady 最原始长成 stedy 这个样子，很明显只是母音混淆的结果，
           但是后来在中古时期转为 steady 就是受到 instead 的影响了。
           instead 发倒３的音比发 instad 蝴蝶音好发音的多，
           因此变音，而为什么只有 steady 受到期影响而改变拼音，这就不得而知了。
           instead 原意是「站到某事物里头来」，所以是「取代」的意思，而steady 造字也不难，
           跟 constant 意义很相似。可是有些用语还是要小心固定搭配，
           比如说我跟女朋友多年感情稳固，只能说是 go steady，不能说是 go constant。
           相反的，如果对方花心，那叫做 play the field。


22. constitute = con~ (一起) + stitute (站)
               = 使(事物)都站在一起
               = (vt.)构成、形成
               = compose
               = make up

               stitute 其实是从 statute 变来的的一个变形体，
               从 a 变成 i 只是发音更顺畅的考量而已。


23. destitute = de~ (往下) + stitute (站)
              = 站立情况一直往下去(的状态)
              = (adj.)贫困的、欠缺的
              = needy
              = poor

              这个字原本是「某人被排到低级阶层去」，原本是富商，结果倒店了，变成佃农，那适用这个字。
              但是这是中古时期的意义，一旦阶层降低就意味着贫穷，这是后来衍生的意义。
              当然，这个字蛮文言文的，通常指「穷到不行」才会用到这个字。


24. institute = in~ (往内) + stitute (站)
              = 使(事物)矗立在里头
              = (vt.)设立 (n.)学院

              这个字当动词用时，跟found、establish 没啥大差异，只是institute 比较偏向于法律用语，
              如 to institute new laws。这个字跟 institution 很难分，
              前者当名词指学校里头的「学院」和「研习会」，而后者泛指任何「机构」和「公共建筑」。
              研究所可以说是 graduate institute，但是慈善机构只能说是 charitable institution。

25. substitute = sub~ (在其下) + stitute (站)
               = 站在(某物)之下之事物
               = (n.)代替品、候补 (vt.)替代
               = replace

               「在某事物下头」通常指比较不好的事物，从下方站上来上头，这个意象叫做「替代」，
               小心后面接的介系词是for，for 在此并不是「因果」，而是「等值」的概念，
               如：ten NT dollars for two buns。因为「取代」要找个「等值」的事物来，所以用 for 并不难理解吧？

26. superstition = super~ (在...之上) + stit (站) + ~ion (动作结果)
                 = 在(事情发生之前)就接受事物的状态
                 = (n.)迷信

                 在古老的文化中，把某事物、某想法放在某事实之上，看得比事实还重，
                 这叫做「迷信」。这个字演变得很难解，但是基本观念就是把「自我的想法」放在「事实」之上的这个意象。

【注】
~sta 是个非常大的字根，除了以上介绍的这些字，还有一些变体字，如 obstinate、system，
这个字根的单字难度非常高，且诸多意义演变常出人意表，需多复习、多思考。
```
