字根三：~tract = draw = 拉
===

```
1. attract = at~ (ad~ 的变体，去使) + tract (拉)
           = 去把人拉过来
           = (vt.)吸引
           = draw

2. contract = con~ (一起) + tract (拉)
            = 把大家拉在一起的事物
            = (n.)契约、合同 (v.)订约

3. detract = de~ (向下) + tract (拉)
           = 把事物的程度向下拉
           = (vt.)减低、减退

4. distract = dis~ (离开) + tract (拉)
            = 把某人拉到旁边去
            = (vt.)使分心

5. extract = ex~ (向外) + tract (拉)
           = 把事物向外拉走
           = (vt.)拔除

           这个字通常只用于「拔牙」，是个很狭义的字，而「拔牙」很多人用错，
           不是：I extracted my tooth. 因为拔牙不可能自己动手，必定请别人动手，
           而是：I had m​​y tooth extracted.
           类似：I had the photos developed. 英文在叙事上比中文重视客观逻辑，比中文简单。

6. protract = pro~ (向前) + tract (拉)
            = 把事物往前头一直拉
            = (vt.)延长
            = prolong
            = elongate

            protract 这个字比 prolong 难用，前者比较狭义，只限用于某几种搭配之中，
            例如：You'd better protract your vacation. 应改用 prolong

7. subtract = sub~ (向下) + tract (拉)
            = 把下头的事物拉走
            = (vt.)扣除、减去

8. retract = re~ (向后) + tract (拉)
           = 把事物拉回来
           = (vt.)撤回、取消
           = nullify
           = withdraw

9. abstract = abs~ (ab~ 的变体，from) + tract (拉)
            = 从现实事物中拉离出来
            = (adj.)抽象的 (n.)摘要

            为何 ab~ 会变体为 abs~ ？因为发音问题，abtract 是很难念，在英文中 tr 合成一个音，
            在拉丁文或是所有欧洲语言中 tr 是发成 t 和 r (弹舌) 两个音的，
            所以 ab~ 碰上后面是 [k] 或是 [t] 的音时会自动变为 abs~，这是合理的发音考量

10. tractor = tract (拉) + ~or (动作者)
            = 用来拉事物的东西
            = (n.)拖拉机

11. tract = 拉
          = (n.)大片区域

          tract 原本是「水流汇集地」，符合「拉」的原意，后来演变成为任何事物集聚的大块区域。
          通常越长、越大的单字越规律、越好解，越简单、越原始的单字越难解。
          因为简短的字根可能和原字根差异非常大（谁能想到 do 的字根是 bha？ ），要多花心思。
```
