字根四：~pose = put = 放
===

```
1. pose = 放(身体)的状态
        = (vi.)摆姿势，(n.)姿势

2. compose = com~ (一起) + pose (放)
           = 把(东西)都放在一起
           = (vt.)创作
           = 把(精神状态)都收回在一起
           = (vt.)使镇静

           这里 con~ 变成 com~ 是因为发音的缘故，后面是 p 的唇音，
           因此所有的 n 都会变成 m，如：imperfect、compress 等等。

3. depose = de~ (向下) + pose (放)
          = 把(人)往下丢
          = (vt.)罢黜

4. dispose = dis~ (away) + pose (放)
           = 把(事物)放到一边去
           = (vt.)排列 (vi.)处理

           注：名词 disposition 的意义有改变

5. expose = ex~ (向外) + pose (放)
          = 把(事物)往外放
          = (vt.)暴露、揭露

          注：expose 有两个名词：exposure 和 exposition，意义不同。

6. impose = im~ (in~ 的变体，向内) + pose (放)
          = 把(事物)放进某人心中
          = (vt.)强加于、打扰

7. propose = pro~ (向前) + pose (放)
           = 把(事物)放到(人们)之前
           = (vt.)提议、求婚

           注：英文造字极度有趣，propose 跟「求婚」有啥关连？当然有，
           求婚时，西方男子一定会把戒指摆在女方眼前。
           另外，名词 proposition 意义有些变化，要留意。

8. repose = re~ (向后) + pose (放)
          = 把(身体、事物)放到后头去
          = (vt.)使安息、使寄托

          这个字有个很有趣的衍生字，叫做 repository，中文叫做「储藏室」，
          想一下字根，不得不佩服造字之奇妙。

9. suppose = sup~ (sub~的变体，在其下) + pose (放)
           = 在(事物现象)之下的基本物
           = (vt.)假设、认定

           西方人认为所有的事物、道理都是建筑于假设、认定之上，那么当然假设是在事物现象之下！
           suppository，医学用词，栓剂，坐药

10. oppose = op~ (ob~的变体，对着) + pose (放)
           = 摆在某事物的对面
           = (vt.)反对
           = object to

           oppose 这个字很有趣，ob~ 原本是「toward」的意思，「朝着某人或某事物把东西放着」，
           可以想像一下菜市场或街头抢生意的情况，op​​pose 的精髓不在于「对抗」，而在于「相对」，
           从这个字的衍生字opposite、opposed 都可以看出这一层意义，其实正确的解释并不是「相反」，
           而是「相对立」，这两个词并不完全相等。the opposite sex 称为「异性」，
           而 at opposite ends 称为「截然相反」，仔细检验：这两个词中的 opposite 是不同义的。

11. transpose = trans~ (across) + pose (放)
              = 把(事物)从这头放去那头
              = (vt.)调换位置

12. juxtapose = juxta~ (靠近) + pose (放)
              = 放(事物)靠近放在一起
              = (vt.)并列放着

              juxta~ 是一个非常罕见的字首，在拉丁文中 juxta 的原意是「靠近」的意思，
              被英文借去变成了just ，just 在英文中不就是「就是、接近」的意思吗
              至于 just 变成「公平」的近代含意，都是由「靠近」最原始的含意衍生出来的。
              接近行事道理，所以称为公平、正义。

13. interpose = inter~ (between) + pose (放)
              = 放在两者之间
              = (vt.)使插入、介入

14. position = posi (放) + tion (动作状态)
             = (东西)放在某处的状态
             = (n.)位置、职位、情势

             position 为什么由 pose 变成 posi 也纯然是发音方便而做的调整。

15. apposition = ap~ (去使) + position (位置)
               = 去达到某事物的位置
               = (n.)文法中的同位语

               这个字有个更常用的形容词，叫做 apposite

【后记】事实上，post 这个字根也是来自于 pose 这个字根（放在一个地方的
        标志物，所以叫做「竿，柱」），当然每个字根几乎都有几个变形字，
        只因为来源不同，~pose 来自于动词原型，而 ~post 来自于过去分
        词，需深入学习。
```
