字根七：~port = ~fer = carry, bring = 携带、搬运
===

```
1. port = (人们)携带东西(的地方)
        = (n.)港口

        port 从「携带」演变为「港口」是可以理解的一个「形象」问题。
        port 和harbor 的区别：
        harbor 这个字来自于德文的harbingar，
        这个字的字源已经不重要，但是在德文中这个字有「保护」的意思，
        因此harbor 的意义著重于「天然避风港」，而port 的意义著重
        于「商务进行处」，因此风景区偏向于使用harbor，港口市镇（如London
        等）比较偏向于用 port。

2. porter = port (携带) + ~er (行为者)
          = 做携带搬运工作者
          = (n.)挑夫、搬行李工人

3. portfolio = port (携带) + folio (叶子)
             = 可携带在身上的叶片
             = (n.)文件夹、一批文件、投资组合

             古人有段时间传递资讯就是把叶子带在身上，如同今天的文件
             引申到投资公司或是股票公司时，相当于不同的投资搭配或是证券的组合。
             由于股票风行，因此这个证券行办公室内的用语竟然流行成大众用语，
             泛指个人或是公司对于不同股票或证券的投资搭配决策，如：
             You should go find a good stockbrokerto fix your portfolio.
             「找个好经纪人来调整你的投资策略吧！」

4. portable = port (携带) + ~able (能够的)
            = 可以携带的
            = (adj.)手提式的

            有段时间，手提式电脑叫做 portable computer，
            现在普遍叫做lap~top (computer)，小到可以放在膝上工作。

5. comport = com~ (一起) + port (携带)
           = 携带所有东西(的方式)
           = (vt.)行为举止
           = behave

           这个字在英语中，是个超级文言文，在中古英文时，这个字的意思是「承受」，
           完全符合原本字源，那为什么后来会演变成「行为举止」呢？
           因为 comport 一定要加上反身代名词，如：
           She comported herself well.，
           在古人的眼中，能够成功地「承担自己的情绪、个性、缺失」的人就是优雅的人。
           所以「成功地承担自己」的人称为「举止合宜之人」。

6. report = re~ (一再，往后) + port (携带)
          = 把(好的东西)带回来
          = (n.、v.)报告

7. export = ex~ (向外) + port (携带)
          = 把(事物)携带出去
          = (vt.、n.)出口

8. import = im~ (in~ 的变体，向内) + port (携带)
          = 把(事物)携带进入
          = (vt.、n.)进口

          important 的确是由 import 演变而来的，因为自己缺乏，
          所以才需要从外头携带回来，因此就是「重要」！

9. deport = de~ (向下) + port (携带)
          = 带(某人)从下头出去
          = (vt.)将...驱逐出境

          de~ 通常有使某人事物进入一个较糟、较低下的状况，
          「往下」可能是实际上，也有可能是「抽象上」的，拆招时务必留意。

10. support = sup~ (sub~ 的变体，在下) + port (携带)
            = 从下方携带东西(往上) = (vt.、n.)支持、支撑

            de~ 和 sub~ 都跟「下」有关，但是留意前者比较趋近于动态「向下」，
            后者比较趋近静态「在其下」或「从其下」。

11. transport = trans~ (across) + port (携带)
              = 把事物从这头带去那头
              = (vt.、n.)运输

12. opportune = op~ (朝向) + port (携带) + ~une (~ion 的变体，动作状态)
              = (在某时候)把(事物)带过去的状态
              = (adj.)恰当、合时宜的

              这个字原意是「在我需要时送东西过来」这个意象，所以演变为「恰当的」。
              而opportunity 也是根据同样的意象而衍生的，所以叫做「偶然的时机」。

13. ferry = ferr (携带) + ~y (动作结果)
          = 携带(人事物)的动作
          = (n.)渡轮、渡船 (vt.)运送

14. confer = con~ (一起) + fer (携带)
           = 大家一起带出(意见)
           = (vi.)协商、授与

           ~port 和~fer 意义一样，但是在造字的过程中，同样的字根配上同样的字首还是会产生不同的结果，
           这是字源学上很令人挫折、也很令人感兴趣的地方。
           第二个意义「授与」是后来衍生的意义，大家一起把东西拿来给你，那自然是「正式的授与」。

15. differ = dif~ (dis~ 的变体，离开) + fer (携带)
           = 把事物带离开
           = (vi.)差异、有区别
           
           看到字源之后，明白了为何 differ 后面常接from。

16. prefer = pre~ (在之前) + fer (携带)
           = 把(某事物)带到之前来
           = (vt.)偏爱、宁愿选择
           
           to prefer A to B 是个很有名的「公式」，可是从来没有哪一本文法书解释过为什么。
           由字源来看，这是「把A 放到B 之前」的意思（我放在前面表示我偏爱），
           所以这里的to 是表示两者之间的位置关系：
           对于 B 而言，A 在前头。

17. infer = in~ (向内) + fer (携带)
          = 把(自己意见)带进事物中
          = (vt.)推断、认定
          = make a judgment
          
          很多人以为infer 等于imply 是大错特错的，infer 通常是读者在做的事，
          而imply 通常是作者在做的事，两者意义几乎是相反。

18. defer = de~ (向下) + fer (携带)
          = 把(事物)往下带(而未往上)
          = (vt.)延迟、延缓
          = postpone
          = delay
          
          从 defer 和deport 两字之分就知道，其实字源解析法并不能解决所有的单字问题（没有一种方法能）
          教字汇至少有七八种方式，除了字源之外还有字辨法、谐音法、拟声法、近义反义记忆法...等等
          但是没有一种方式能够解决所有的问题，方法在个人，必须要运用大量的想像去解决问题

19. offer = of~ (ob~ 的变体，朝向) + fer (携带)
          = 带(事物)给(某人)
          = (vt.)提供、给予
          = provide = afford
          = (n.)提议、出价
          
          从这里的 ob~ 变成 of~ 可见英语经常为了发音顺畅更改字根
          此外也有很多英文外型不像，但是意境像。

20. refer = re~ (往后) + fer (携带)
          = 回头地去带(事物)过来
          = (vi.)参阅、提及 (vt.)提交
          
          refer 这个字其实很困难，因为它及物和不及物时的用法差很多，
          原因在于re~ 这个字首本身意义就有些模糊。现在做的事情先停下来，
          回头去找事物，所以衍生出不及物的用法，后面加 to 也是合理的搭配（朝向...找）。
          但是当及物时，refer 的意义是「指引 A 到 B 处去」，又由于 re~ 是往后的意思，
          因此暗喻：A 原本属于 B 处，只是让 A 回归到它该去 的B 地方而已，如：
          The shop referred the complaint to the makers.
          My professorreferred me to some articles about this topics.

21. transfer = trans~ (across) + fer (携带)
             = 把事物从这头转带去那头
             = (vt.、n.)转移、调动
             
             transfer 指人事物状况的转移，而 transport 则指人事物的位置移动，
             一偏向抽象概念，一偏向具体事实。a transfer student 叫做「转学生」，
             有一个字叫做metaphor，中文称为「暗喻」，这个词其实和 transfer来自于完全相同的字源：
             meta~ (across) + phor (~fer 的希腊文体，携带)，
             换言之，metaphor 是个希腊文版的transfer，把事物从这头转带去另一头，
             因此在文学中的手法，这称为「转喻」或「暗喻」，比如：
             My love is a red rose.
             这就是把「爱」这个概念转带过去「红玫瑰」的意象之中。

22. suffer = suf~ (sub~的变体，在其下) + fer (搬运、携带)
           = (人事物)在下方承受、搬运东西
           = (vi.)受苦、变糟 (vt.)遭受
           
           很明显，这个字是从古早时候低下的搬运工人的意象演变而来。
           这个字要小心及物与不及物用法上的差别，
           当不及物时，后面的介系词常用from，如：
           Mymother suffers from headache.
           这里的from 是表示「受苦」的源头、起因，
           从字源上来看，这个from 是意味着搬运的东西从上往下堆在人身上。

23. proffer = pro~ (向前) + ffer (fer的变体，携带)
            = 把(事物)拿​​向前
            = (vt.)端出、递出
            
            这里会多出一个 f 的原因是因为要把重音节留在第一音节 pro 之上，
            因为落在第二音节fer 上头会不好念，因此加上一个f 把 pro 和 fer 分隔开来，
            使得那个 o 形成「子母子」的形式，让人一看字就比较能确定重音应该在 prof。
            这个字很文言文，一般我们都用 hand、give、pass 等这些好用的简单字。

24. circumference = circum~ (around，环绕) + fer (搬运) + ~ence (性质)
                  = 带着事物在外头绕一圈的事= (n.)圆周

25. interfere = inter~ (between，从其间) + fere (~fer的变体，携带)
              = 把(事物)带进两人(物)之间= (vi.)干涉、妨碍
              
              为什么好好的 ~fer 字根到这里变成 ~fere 呢？是的，发音困难。
              interfer 原本发音落在第一音节in 那里，可是inter 在念很快的时候会使得[t] 音产生弱化
              （如：little、cattle...等等），这样一来，整个字语音就变得很模糊，
              为了发音清晰，重音节往后移，落在~fer 上面，可是interˊfer 也很难听，只好把 ~fer
              的音加长，字尾加上~e，原本的卷舌音转为 [ir] 就很好念了。interfere 是个不及物动词，
              后面介系词接in、between、with 皆可，意义稍稍不同，但 with 比较常见。

26. fertile = fer (携带) + ~t + ~ile (~ible的变体，能够的)
            = 能够携带(东西)出来的
            = (adj.)土地肥沃、丰饶的
            
            这个词原本指土地能够源源不断地产生（带出）东西出来，所以土地才会丰饶，
            但近年来也有一种蛮新的用法，是指人能「源源不断地想出主意」，就是像「智多星」那种思考能力。
            相反词 infertile 除了指土地贫瘠之外，还有一个重要用法，就是指女人「无法生育」。
            而 fertilizer 则是「使土地能生产之物」，叫做「肥料」
            中间多个t 是什么意思？是为了方便发音，不信把那个 t 去掉试试看，
            而发音时要注意，fertile 中的 t 并不是发[t] 音，而是发跟 little 一样的那个滑音
            而字尾 ~ile 是很常见的，减掉 b 那个子音，让整个字念起来更顺畅，
            可是字尾 ~ile 的发音很麻烦，因为如 fertile 和 mobile 这里的i 都是不发音的，

【后记】其实，在字源学家眼中看来，~port、~fer、~phor 三个字根都是来自于同一源头的，
        因为[p] 跟[f] 音非常接近（如同[b] 跟[v] 一般），所以模糊念起来的话
        （希腊文中，p 和h 是都发音的），您会发现的确这三种形式最原始时一定没啥分别的，
        您也会很明白为什么希腊文的ph进到英文里头全部都变成了 f 的音。
        很多事实说穿了其实不值一哂，但是整个思考的过程才是真正研究必须要有的精神。
        「转音」在语音学上是一门专业的学问，我们不需要去精研才能得知目前得到的结论，
        
        字根字源学习法还有一个很有趣的地方就是：怎么去拆字？这是很难的学问，
        一般人看到 abound 应该都会拆成ba~ + bound，可是这是错的，应该拆为 ab~ + ound。
        几乎每个学过 ~port 这个字根的人都会以为 portion 应该是跟 ~port 相关的字（port + ~ion），
        可是很抱歉 portion 这个字跟 ~port 这个字根一点关系都没有。
        碰到英文中无法用字源解析法去分析的单字时，学习者该坦然，
        而去寻求其他方式来解决问题，这就是语言、也是人生。
```
