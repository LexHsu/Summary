### 状态码分类

|     状态码    |   已定义范围   |    分类    |
|:------------:|:------------:|:----------:|
|     1**      |   100 - 101  | 信息提示    |
|     2**      |   200 - 206  | 成功        |
|     3**      |   300 - 305  | 重定向      |
|     4**      |   400 - 415  | 客户端错误   |
|     5**      |   500 - 505  | 服务端错误   |
