字根二：~ject = throw = 丢掷
===

```
1. abject = ab~ (from) + ject (丢) = 从某地把(某人)丢出去
                                   = (adj.)悲惨的
                                   = poor = miserable

2. object = ob~ (toward) + ject (丢) = 丢东西的事物= (n.)物体
                                     = 朝着丢的对象 = (n.)受词、目标
                                     = 大家朝着事物丢东西 = (vi.)反对

    ★这个字的词类变化非常重要。从字根来看，就知道 object 当「反
      对」时，后面的介系词为什么一定要用to 了~~ 这样学英文不才是
      轻松、合理吗？ oppose 跟object 不同，前者是及物动词，后者是
      不及物。

3. subject = sub~ (在下方) + ject (丢) = 把(某人)丢到脚下= (vt.)使臣服
                                 = 把(事物)丢到下方(征服它) = (n.)科目

    ★这个字的名词字义演变非常之大，背后故事暂且不提，要想办法记喔。

4. deject = de~ (往下) + ject (丢) = 把(某人)往下丢= (vt.)使灰心沮丧
                                                    = depress
                                                    = discourage
                                                    = disappoint
                                                    = let (sb.) down

5. inject = in~ (往内) + ject (丢) = 把事物丢进(人体)内= (vt.)注射

    ★inject 是个正式用字，to give sb. a shot 才是常用口语。

6. eject = e~ (ex~ 的变体，向外) + ject (丢) = 往外丢= (vt.)喷出

    ★eject 比exject 好发音得多，这是变体的主要原因，这个字也是正式
      用语，这个字有个亲戚，叫做ejaculate（e~ + jacul + ~ate），其
      实也是ject 家族中的一员，只是外型变化有点大，其变形原理牵扯到
      拉丁文本身变音构字原则，因此仅提出参考，不做介绍。

7. reject = re~ (一再、向后) + ject (丢) = 把东西往回丢= (vt.)拒绝
                                                        = refuse
                                                        = decline
    ★reject、refuse、decline 其实意义不尽相同，此处的「=」只是表示
      「近义词」。

8. project = pro~ (向前) + ject (丢) = 往前丢出= (vt.)投射、伸出
                                = 丢在人前(方便讨论)之事物= (n.)企画

    ★从字根就可看出project 和plan 用法上不同之处。

9. adjective = ad~ (去) + ject (丢) + ~ive (本质属于物)
                                   = 把本质上相关物丢过去
                                   = (n.)形容词修饰语

    ★形容词是用来加在名词之前或后，加以说明其性质的一种东西，
      只要多用点想像力，背单字就会是件快乐的事情。附注：~ive
      本来是个形容词字尾，用作名词是长期误用的结果。

10. conje​​cture = con~ (一起) + ject (丢) + ~ure (动作状态)
                                   = 大家一起丢出(意见、想法)
                                   = (n.)猜测

    ★名词字尾~ure = ~ment = ~tion 意义都差不多，都是表示动作的过程
      或是产生的结果。如果是~ance = ~ence 则比较偏向于表达「事物的
      本身性质」名词字尾。

11. jet = ject 的变体 = 丢出 = (n.)喷射机


12. traject
```
