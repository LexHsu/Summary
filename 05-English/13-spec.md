字根十二：-spec = -spect = -spise = = -scope = -vid = -vise看
===

```
1. species = spec (看) + -ies (-y 的复数，事物性质)
           = 看到的事物性质
           = (n.)种类、物种

           species 是 specy 的复数，原意是「看到的事物」
           比 things、stuff 都还文言，泛指生物学上生物的种类（从外观上得知异同处）。


2. special = spec (看) + -ial (具有...性质的)
           = 具有某事物外观上特质的
           = (adj.)特殊的、独特的
           
           special 是 species 的形容词，各个物种的特性，所以「特殊而相异」。
           especial 只是前头再加个 e-，也就是 ex- 的变体：「事物外头所看到的形式」，
           慢慢跟special 同义，由于收到法文的影响:
           形容词时 special 比 especial 常用，
           而副词时 especially 比specially 常用。


3. specify = spec (看) + -ify (使得)
           = 使得(某人)能清楚见到
           = (vt.)具体指定、详述
           = clarify
           = make clear
           
           很多人认为 clear 就相当于 specific，其实不然。
           clear 常强调「自然的」清晰明白，而 specific 常指「人工的」使清晰明白：
           所以 specific 可以指「特有的」，而 clear 可以指事物「清澈的」。


4. specimen = spec (看) + -i- + -ment(-ment 的变体，事物结果)
            = 用来看的事物结果
            = (n.)样本、标本
            = sample


5. conspicuous = con- (共同) + spic (spec 的变体，看) + -uous (多...的)
               = 大家都能看见的= (adj.) = 明显的、引人注目的
               = outstanding = remarkable = striking = obvious

  ★conspicuous 要比conspecuous 的倒３好发音，是慢慢从e 演变为i 的
    原因，至于为什么是-uous 而不是-ious，我想可能是为了要和specious
    做分类区别吧，当然只是猜测而已，这种字尾演变可能要从语音学下手啰！
    在中文里的「大放异彩」，英文是写做to cut a conspicuous figure。

6. speculate = spec (看) + -ul- (-ule 的变体，小事物) + -ate (使得)
             = 眼睛都往小事物去看= (vi.)猜测、投机取巧

  ★-ule 其实并不是太冷门的字尾，因为capsule、formula 都还算是常用
    字，但是看过就够了，无须深究。在很多字中，-ul 只是方便发音而已。
    speculate 和speculation 在古早时候只是用作于「看到小事物而做猜
    测」，这几年房地产、股票盛行，这个字也大红，to do sth. on
    speculation 都叫做「投机」，在各经济杂志风行不已呢。

7. spectacle = spect (看) + -acle (器具) = 用来(大家)看的事物
                                         = (n.)眼镜、奇观

  ★最早spectacle 这个字是做「眼镜」解，古时候泛指帮助看东西的事物，
    没想到玻璃镜片盛行，竟慢慢被glasses 干掉，它心底一定颇为不甘吧？
    当作「壮观事物」是比较近代的演变，我常开玩笑：「需要大家戴起眼镜
    来看的事物」一定是壮观、稀奇的事物。
    不过，如果某个人make a spectacle of himself，可不是造就什么壮观
    场面，而是闹出一些糗事让大家戴眼镜来看，这叫做「出洋相」。
    至于spectacular 和spectator 不过都是词类变化而已。

8. specter = spect (看) + -er (人事物) = (心中)所看到的事物
                                       = (n.)幽灵、亡灵

  ★这个字超级文言，不记也罢，因为​​「鬼怪」的单字实在太多，如ghost、
    apparition、vampire、zombie...等等，除非写做论文，不然实在无须
    用到这种单字。

9. expect = ex- (向外) + spect (看) = 向外(一直)看= (vt.)期待
                                                   = look forward to

  ★expect 用法复杂，需要查字典，后面接人事物皆可，要小心它跟hope 不
    同，我们说to expect your letter，不说to expect for your letter，
    那个for 是多余的，不要被to hope for your letter 所影响。
    至于这个字作女人怀孕解，是衍生的意义，可是也很贴切，不是吗？ ：）

10. inspect = in- (向内) + spect (看) = 往(事物)内部看= (vt.)检查
                                                       = examine = check

  ★inspect 这个字比较文言文，通常指规模较大的检测，如a tour of
    inspection 叫做「考察团」，这个词无法代换。

11. respect = re- (一再) + spect (看) = (某人事物需要)一再去看
                                      = (vt.、n.)尊重
                                      = esteem = look up to

  ★这个字不难，当名词时，后面介系词常用for，如：to show respect for
    my teacher，这里的for 当然是「因果」和「朝向」，不能用to -- 如果
    用to 会变成「把respect 这事物拿给我的老师看」了。

12. aspect = a- (去使) + spect (看) = 去看(人事物)的结果
                                    = (n.)见解、方面 = respect

  ★这个字中文的解释群中，我觉得「见解」最生动，既兼顾字源又涵盖字义，
    实为良译！ 「看到人事物而得到之结果」谓之「见解」-- 这个字古早的意
    思保留在「外表、容貌」这个意义之中，可惜这层意义已经慢慢被时代淘汰
    掉了，现代人都用appearance 这个字啰。
    在in xxx aspect 这个片语之中（当然用in），respect 是可以用来代换
    的，respect 表示「与事物的关系」是很古早的意义，由于「一再地看」因
    此产生「对事物的见解」，这也只存在于很古雅的文句中，比如说with
    respect to、in respect of、respective...等这些字几乎都只出现于正式
    文件之中，所以一般学子把respect 归类于「尊敬」，把aspect 归类为
   「见解」就足够了，只需要知道in xxx respect 也是正确的，如此即可。

13. prospect = pro- (往前) + spect (看) = (人们)往前看(事物)的结果
                                        = (n.)景观、前景、预期 = vista

  ★同样地，大家会发现「前景」这个中译翻得实在太漂亮了（只可惜英汉字典
    中这样的佳译实在太少了）。要小心的是：当「景观」用时是不可数名词
   （如同scenery 一般），而当「前景、预测」之时，我们常用复数，如：
    Prospects for her promotion seem bright / good.在这句中，prospect
    的意义跟future 根本不同，但是却听到很多英文老师说可以代换，怪哉。
    至于vista 则是个文言用词，比不上prospect 的口语。

14. retrospect = retro- (往后) + spect (看) = 往过去回头看
                                            = (n.)回忆、回顾

  ★retro- 其实并不是很罕见的字首，像现在一片流行复古风的时候，喇叭裤、
    紧身衣、迷你裙又再度踏上舞台，这种风气就叫做retro，而偏爱复古风的
    人叫做retrophile（-phile 是「爱」的意思）。
    再者，retrospect 并不等于memory，小心其字源：In retrospect, all my
    happy past days seem like just yesterday. ​​这里可无法代换。

15. suspect = sus- (sub- 的变体，在下) + spect (看)
            = 由下往上一直看= (vt.)怀疑、推测(n.)疑犯

  ★这个字蛮象形的，颇像我们思考时总习惯于把脸朝天去思考、去推测，不是
    吗？所以可知当人们心中不确定而思索之时，各民族的人应该都差不多的，
    至于名词的用法当然是后期衍生出来的。
    当然，suspect 后面接「事情」，而不可能接不定词，这是可以推理的，
    因此：to suspect his lying = to suspect him to lie。

16. despise = de- (向下) + spise (spect 的变体，看) = 往下看(人事物)
                                                    = (vt.)鄙视、蔑视
                                                    = look down on

  ★despise 和despite 是同一个字源演变出来的，后者已经变意，从「轻视、
    忽略」演变为「不顾」：He went ahead despite my warning. -- 他藐视
    我的警告而一意孤行-- 并不见得不合理。
    而那个 spise 是从何而来的呢？在拉丁文中，「看」这个动词原形长成
    specere 这个样子，而spectus 只是它的过去分词型态，而spect 却比
    spec 常见，因为在印欧语系的语言之中，动词原形（infinitive）并不
    常见，熟悉法文或德文的人应该了解这句话。 spise 是从 spec 这个动词
    原形演变而来的-- 不过这个知不知道对于背单字没太大的助益了，只是
    给有志之士参考罢了。

17. espy = e- (ex- 的变体，在外) + -spy (-spise 的变体，看)
         = 在外头看到= (vt.)察觉、发现= discover、find

  ★这个字很文言文，所以认得出即可。有人会问：那 spy 呢？难道不是相
    关字？问得真好，当然是！ spy 是从法文的espier 变来的：「在外头
    张望的人」，完全符合字源规律，只是拼字有变异，因此提出供参考而已。

18. scope = 看 = (n.)眼界、范围

  ★scope 这一串字根是由希腊文演变而来的，拉丁文是spec 而希腊文是
    scope，这可真有趣的呢，两个子音互换一下摆明就是同一个字根咩！
    scope 原本指「所能看到的范围」，衍生为「活动或研究的范围领域」。

19. microscope = micro- (小) + scope (看) = 看小事物的器具
                                          = (n.)显微镜

  ★可别看字源，以为是「小镜子」的意思喔！另外放大镜不叫做 macroscope，
    纵然macro- 是micro- 的相反词，有macroscopic 这个字是没错，但是
    跟「放大镜」可没关系，「放大镜」叫做magnifier。

20. telescope = tele- (远) + scope (看) = 看远处事物的器具
                                        = (n.)望远镜

  ★很多学子自从学了telephone 和television 之后，一直以为tele- 叫做
   「电」，可差得远了呢，tele- 是一个希腊字根，表示「远」-- telephone
   叫做「听到远处的声音」，television 叫做「看到远处的事物」。
   望远镜有两种，一种是单管的，用来看星星、夜景的，比较贵的，那叫做
   telescope，另一种是双管的，用来听歌剧、戏剧时常看到贵夫人拿的，比
   较便宜的，那叫做binocular -- 可别混淆呢！

21. periscope = peri- (四周) + scope (看) = 用来看四周事物的器具
                                          = (n.)潜望镜

  ★小心很多学子都以为「潜望镜」是由下往上看的，叫做subscop​​e，听起来
    似乎合理，但periscope 更能说明其用途呢，不是吗？

22. skeptical = skept (scept 的变体，看) + -ic (相关的) + al (本质的)
              = 去看事物本质的 = (adj.)挑剔怀疑的

  ★这个字看起来很突兀，可是这是美式拼法，英国人拼成sceptical，这就
    很明显了吧？很多时候，英国英文比美式英文更接近古意，这是事实。就
    像大陆的北京话和台湾的国语一般。

23. video = vide (看) + -o (第一人称单数动词字尾) = 我看到(的事物)
                                                  = (n.)录影带、电视影像

  ★video 来自于拉丁文的videre = to see（动词原形），而字尾-o 是 I
    的字尾变化并不突兀，只要对西文、义文熟悉的读者都可以了解。它跟 audio
    的来源一样，audio = I listen。
   「录影带」这个意思反而是后来的衍生，video games 更符合字源！

24. divide = di- (dis- 的变体，分离) + vide (看) = 把事物看分离
           = (vt.)分隔、分配 = separate = group

  ★di- 这个字首有两个意思，第一个是dis- 的变体，另外，在希腊文中，
    数字「二」也叫做di-（拉丁文叫做bi-），反正这两个解释在这个字之中
    都可以自圆其说。这个字最重要的用法不外乎to divide A into groups，
    那个 into 表示「结果」，并不罕见。
    division 只是词类变化，但是在一家公司里头division 到底是什么单位，
    是「部」、「处」、还是「分店」...？这可需要大伤脑筋啰！

25. provide = pro- (往前) + vide (看) = 预料到(事件而先做准备)
                                      = (vt.)供应、备用、规定

  ★provided that 是个很古雅的片语，很多人都知道这个词相当于if，但却
    很少人知道其来源 -- 这里就是了。 Provided it rains tomorrow, ...
    叫做「预料到明天会下雨的情况，那么...」，在这个片语中反而保留了最
    古早字源的含意。至于「供应」虽是后来的衍生意义，但是原本的意象：
   「顾及即将来临的事件」而互相补给食物，早期农村的形象油然而生，虽然
    这种景象在我们社会中已经不可能重现了，但是看字源而思故人，这是不变
    的。

26. evident = e- (ex- 的变体，在外) + vid (看) + -ent (事物本质的)
            = 从外头可看见事物本质的= (adj.)显然的、明显的
            = apparent = obvious = conspicuous

  ★这个字的名词evidence 反而更红，叫做「证据、迹象」，这个字知道的学
    生很多，但用错的更多-- 请注意，evidence 有百分之九十五以上的情况下
    是不可数名词，不用复数！ 「有迹象显示」叫做There is evidence that...，
    没有复数。 Her silence is evidence of anger. 这句也没有复数，切记。
    而且It is evident for her to flee.这是错句，因为evident 是「外观看
    起来」的字根，所以介词搭配用to：It is evident to everybody that she
    will flee. -- to 是表达「相对关系」，在这里才是正确的选择：文法需要
    思考，而不是死记，这是唯一的座右铭。

27. providence = pro- (在...之前) + vid (看) + -ence (事物本质)
               = 在(事件发生)之前即可预见= (n.)先见之明、天意、天命
               = God = fate

  ★这在中文是个文言文的字，在英文中倒还好，因为宗教对于东方人的影响
    并不显著，宗教在西方世界有时几乎等同于最高的法律，因此宗教词汇在
    英语中可谓比比皆是，每个西方知识份子对于圣经必然都有适度熟悉的，
    无论他信不信基督教，这点基础常识必须要有。
    形容词provident 和improvident 比较接近古意，这两个字反而都跟宗
    教无关，因为providence 本来也不是宗教的字，只是后人把「先见之明」
    归因于宗教、神所赋予人类的「特殊力量」，所以才衍生出最后的「天命」
    之意。

28. visage = vis (看) + -age (集合物) = 看到(人)的结果集合
                                      = (n.)容貌 = countenance

  ★这个字字源很简单，但是却很罕用，只是来自于法文visage（脸）的一个
    文学文雅用字而已，知道即可。
    不过这个字却有一个衍生字，叫做vis-a-vis = face to face，也是来自
    于法文的一个缩写词，但这个却常用多了。

29. visible = vis (看) + -ible (能够的) = (adj.)能见到的
                                         ←→ invisible

30. vision = vis (看) + -ion (动作结果) = 看到的结果
                                        = (n.)视力、幻象、洞察力

  ★这个字字源也不难，但是这个字跟sight 的字义却是难分难解，其实这两
    个字非常好分，除了「视觉」之外，这两个字几乎都不大一样。 vision 凸
    显抽象的「见到之事物」，如：a man of great vision、the failure
    shattered my vision、the girl's image appeared in my vision -- 以
    上这些例句都无法以sight 来替代，因为sight 更强调具体的「见到之事
    物」，因此有「视野、见解、景象」这些意义，例句：Mt. Ali is a
    beautiful sight、In my sight, you did a great job、I caught sight
    of the boat....不管怎么变，读者会发现到「抽象」和「具体」这两个
    概念可以判断九成九以上的片语-- 这两个字并不是都能代换的！
    至于 visual 只是词类变化而已。

31. advise = ad- (去使) + vise (看) = 去使得(某人)认清(事物)
                                    = (vt.)忠告、劝告 = suggest

  ★advise 意义已然很明显，但是不晓得为什么还有学生弄错-- to advise
    人to do sth. 是合理的，可是「to advise 人事」是不合理的，这个
    字不能接双受词，要用to advise 人of 事-- 为什么用of 呢？看看
    字源吧-- 去使得某人看清他原本没看清的事物，因此这个事物和他这
    个人的关系是「本质属于」的of -- 难道不对吗？

32. revise = re- (一再) + vise (看) = 对(事物)一再察看
                                    = (vt.)校订、修正

33. devise = de- (往下) + vise (看) = 往(事物)的下方底层去察看
                                    = (vt.)设计、策划 = think out

  ★devise 的意思跟design 非常接近，但前者意义著重在于「发明」，而后
    者意义著重在于「设计」，所以：We devised how to prevent water
    pollution. -- 意义偏向是「想出对策」，而不是纯粹做计画而已。
    这个字的名词device 可真是常用的了，只要是「发明出的小配备、器具」
    或者是「对策」都叫做device -- 注意常指小事物，比如说电话答录机是个
    a helpful device，但不能称为equipment，因为后者指「大型而整体」的
    配备。

34. supervise = super- (在...之上) + vise (看) = 在(人们)之上监看
                                               = (vt.)监督

  ★supervise 常用于老板监督员工的用词，如果是老师监考学生，绝对不是用
    这个字-- 已经看过n 个英文老师犯错了--「监考」叫做proctor。这个
    字跟manage 也不一样，前者重于「监督」，后者重于「提出政策来管理」。

35. improvise = im- (in- 的变体，无) + pro- (在...之前) + vise (看)
              = 之前并未看过(就做某事) = (vt.)即兴创作

  ★这个字比较偏向于艺术，七零年代的摇滚和九零年代的摇滚最大的不同便在
    于当时的乐团，如King Crimson、Pink Floyd、Black Sabbath、Led
    Zepplin 等都着重于improvise，凸显团员的奇才以及嗑药后的灵感迸发，
    当时那种生气蓬勃的演奏可惜今日无可复见！每次看到这个字都会令我想起
    这几个我深爱过的乐团。
    这个字是动词，而且不太适用于日常生活口语中。口语中如果说「即席演讲」，
    不是用这个字而是to make a speech extempore、to speak offhand、to
    make an impromptu speech，词性不同，用法也不同，语言层次也不同。

36. visit = vis (看) + -it (去) = 去看某样事物= (vt.)拜访、参观

  ★it- 并不是稀有字根，exit、transit、ambition...这些字都不是罕见字，
    都是 it- 的字根演变。拉丁文的ire 在法文、西文中也是未来式的基本
    字根。很多读者会对于我常举「对法文熟悉者」这个词感到厌烦-- 难到
    我没学过法文就很难接近字根吗？不是这样的，英国在历史上曾经被法国
    统治上千年，如果没有法国人，英文的拉丁化不会这么彻底，如果一个读者
    完全不懂拉丁文，对于字根字源学习法仍然可以有助益，但是只能进展到
    某个地步而已，再也无法进前了，因为在不了解的情形下，所有字根都变成
    了单字而变成需要死记。然而，有些法文基础的读者会很轻易地在法文字汇
    中清晰可见那些字根字源在活跃着，甚至五成以上的英文字是从法文传入英
    国的，这样说不过份，所以用十句话能解说的字源道理，对于一个法文的学
    习者很多时候只要一句话就够了-- 对于外文系的学弟妹，对于英文字汇有
    浓厚兴趣者，我只能说第二外国语选法文是非常重要的。但是，我一直深信
    兴趣和努力才是真正背单字最厉害的利器，这是什么背景知识都比不上的。

37. visor = vis (看) + -or (动作事物) = (借着)看的事物
                                      = (n.)头盔的护面、安全帽舌

  ★visor 不算太冷僻的单字，因为安全帽分两种，一种是无帽舌的（没有前面
    挡风镜的），一种是有的。有些女孩子化妆或是爱美就会偏向于a safety
    helmet with a visor。而很多开车的司机会在汽车前窗加上遮阳版，是的，
    那就叫做 sun visor -- 料不到吧？这是起源于古罗马战士脸上戴的盔甲只
    露出眼睛的那个护面。

★「看」是个非常大的字根，不言可喻。主要是 specere 和 videre 两大支，
   洋洋洒洒三十七个单字解析并不代表所有的字汇-- view、revue、prudent、
   speck...等这些字都没有列入解析，原因一是变异蛮大的，把所有变异字都
   列入解析的话可能篇幅过长而过于专业，实用性并不太佳-- 所以很多更进
   一步的知识只能依赖各位自身的努力和兴趣了，毕竟没有人能够传授所有的
   事物，很多字还有很多有趣的衍生片语和口语，光一个字上课就可以上掉一
   二十分钟不过份，可是学习的意愿始终在于学生自己，这是我个人做老师近
   三年来的体验-- 学生没有意愿要多学些，就算是全台湾第一名师来都没有
   用。而任何学习的意愿都起源于翻查字典和提问题，一切都至此开始。
```
