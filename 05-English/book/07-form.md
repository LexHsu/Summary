字根：-form = -morph = 形式、形状
===

```
1. form = (构成)形状
        = (n.)形状 (v.)形成

        formal 是个词类变化，字尾 -al 表示「相关的」的一个形容词字尾，所以「与形式相关的」叫做「正式的」。
        informal 以此类推。 formality 常用于「客套话」或是官方的总规矩一堆、效率不彰
        如：Let's cut all those formalities and start the ball rolling, shall we?

2. format = form (形式) + -at (-ate的变体，使得)
          = 使得(事物)​​有形式
          = (n.)样式、编排

          -ate 大多数都是动词字尾，而这个字为什么不是 formate 这个正常模式呢，非常有趣。
          因为formate 只有两个音节，因此重音势必落在前方form，而后面的-ate 就变成弱音，
          慢慢地这个字被借来用作名词，formate 看起来就一副动词样，因此变成今天的 format 是应该的。

3. formation = form (形式) + -at (-ate 的简化，使得) + -ion (状态)
             = 使得(事物)​​产生出形式的状态
             = (n.)队形、结构

             structure 是「构造」，formation 也是「构造」
             从字根判断：struct 是「建造」的意思，form 是「形成」的意思，
             因此前者指人工建筑物（如大楼）的结构，后者指自然物（如石头、暴风等等）的结构。
             一目了然。

4. reform = re- (再次) + form (形式)
          = 再次给予(事物)一个形式
          = (vt.、n.)改造、革新
          = improve
          = better
          = ameliorate

5. conform = con- (一起) + form (形式)
           = 大家一起(同意)的形式
           = (vi.)符合一致、顺应习俗

           conform 后面几乎一定要加to，如 to conform to the ISO 9002 safety standards。
           其实很好理解，to 的概念就是一个箭头，去「靠近」大家订的形式，所以才叫做「符合一致」
           「符合一致」这个词在英文里头至少有七八种代换，如 accord with、consist with、
           to be consistent with、to be in line with，只有 conform 后面用 to，
           所以这才需要特别了解和讲解。

6. deform = de- (向下) + form (形式)
          = 形式向下(掉落)
          = (vt.)使畸形、毁坏...的外型

7. transform = trans- (across) + form (形状)
             = 形状由这一头转到另一头去
             = (vt.)使转化、使改观

             deform 和 transform 区别，请看字根：
             英文里头「向下」通常有不好的含意，如 depress、deject、decay...
             因此 deform 是从原本的形式变成较低下、较糟的形式，所以才「畸形」；
             而 transform 是从这一头到另一头去，因此是「变成另一个样子，跟原本大不同了」。
             因此生下来有畸形脚，这是 deform，庄周梦到自己变成蝴蝶，那叫做 transform。

8. perform = per- (through，完全) + form (形式)
           = 使形式完全(展现)
           = (vt.)实行、表演

           per- 是个蛮麻烦的字首，原本是「穿过」，演变成「完全」，
           中文颇难传达这种概念，「穿过、经历很多、各种形式」，所以才有
           「执行」这层意义，「表演」是比较后来的意义。

9. inform = in- (向内) + form (形式)
          = 把形式传进(某人体)内
          = (vt.)告知

          很多人说 inform 后面只能用of，这是不对的，inform 后面加about、on、against 都对，
          只是意义上的差别就需要读者自行查字典了。 well-informed 是个很漂亮的复合字，
          叫做「消息灵通、见多识广的」。顺便带一个超漂亮的句型，to keep sb. informed
          -- Newspapers always keeps me informed of the latest events.
          太正点的句子，比什么 tell me 什么的好太多了。
          老板叮咛一句：Keep me informed, OK? 就是要妳随时把最新状况回报给他知道

10. uniform = uni- (单一) + form (形式)
            = (规定的)单一形式
            = (n.)制服 (adj.)一贯不变的

11. morphology = morph (形式) + -ology (研究、学问)
               = 研究(文字)型态的学问
               = (n.)词法、构字学

               -form 是拉丁字根，而-morph 是希腊字根，英文里通常拉丁字根配上拉丁字首，
               希腊字根配上希腊字首（还是有少数例外先不提），但是一般大众对于拉丁字首较熟悉，
               对于希腊字首就很不熟悉，所以要小心留意跟 -morph 搭配的一些字首字尾，
               像 -ology 就是一个希腊字根。

12. metamorphosis = meta- (across) + morph (形状) + -osis (状态)
                  = 事物的形状从这一头转化到另一头去
                  = (n.)变形 = transformation

                  meta- 是一个希腊字首，意义跟 trans- 这个拉丁字首一样，
                  所以意义当然跟 transform 密切相关，-osis 是个希腊字尾，
                  就相当于拉丁的 -ion​​ 或 -tion，不过现在大都只能在病名的原文中看到了。
                  要留意这个字只有名词型，而且这个字比较文言文。

13. amorphous = a- (ab-的变体，离开) + morph (形式) + -ous (多...的)
              = 很多形式远离(原本状况)的
              = (adj.)无定型的、杂乱无章的

              ad- 这个字首是「去、使」，而 ab- 这个字首是「from、离开」的意思，
              这两个字首都常因为后面的字根开头字而简化为 a-，
              如：amount 这里的 a- 是 ad- 的变体，但是 amorphous 如您所见，就是 ab- 的变体，
              所以看到 a- 什么开头的字都是很头痛的，不过八成是 ad-，只有一成多会是 ab-，那剩下的呢？
              剩下的什么都不是，如 apple。

              为什么不写成 abmorphous 呢？因为 [b] 跟 [m] 都是唇音，两个音混在一起很难发音。
```
