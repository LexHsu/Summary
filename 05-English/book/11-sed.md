字根十：~sed = ~sid = ~sess = ~sist = 坐
===

```
1. sedate = sed (坐) + ~ate (使得)
          = 使得(某人)坐下
          = (vt.)使服镇定剂 (adj.)沉着稳定的

          有个相关词 sedentary 就是 sed + ~ent + ~ary
          ~ary 是形容词字尾，因此是「与坐相关的」，坐着的，不爱活动的

2. sediment = sed (坐) + ~i~ + ~ment (动作状态)
            = 事物(往下)坐后的状态
            = (n.)沉积物
            = residue
            事物往下坐」不就是「沉淀」吗？那个多出来的字母 ~i~ 只是方便发音而已。

3. supersede = super~ (在...之上) + sede (坐)
             = 坐在(某事物)之上
             = (vt.)取代、淘汰
             = replace

             这个文言文用字字目前几乎都出现于被动语态之中：
             A is superseded by B = A is replaced by B。
             字尾那个 e 只是方便发音而已，~sed 位于重音节，不「长音化」是不行的。

4. assiduous = as~ (去使) + sid (坐) + ~uous (多...的)
             = 让人一直坐着的(状态)
             = (adj.)勤奋不懈的
             = diligent


5. reside = re~ (一再、向后) + side (坐)
          = 一而再(去某地)回去坐
          = (vi.)居住、归属 = live

          这个字常跟 live 和 inhabit 搞混，reside 比 live 正式太多，
          「驻美国外交官」会用 reside 而不会用 live：
          the diplomat residing in the States。
          而 inhabit 本身已经有 in，因此 inhabit 一直是个及物动词，后面无须再加 in。
          reside 不常用，但是 resident 和 residence 却常用，而从 sid 演变成 side 纯粹是发音的考量。

6. preside = pre~ (在...之前) + side (坐)
           = 坐在(人们)之前
           = (vi.)主持、担任主席

           这个字是不及物动词，后面常接 over：to preside over the conference，
           当然用 over 也是可以理解的，凸显「凌驾于、从头至尾」。 「总统」president 来自于同一字根。

7. subside = sub~ (往下) + side (坐)
           = (事物)往下沉坐
           = (vi.)沉降、消退

           subside 这个字本来多指于实际事物的陷落，但现代英文中多用于抽象情绪的「减退」，
           比如说：Her anger subsided into a whimper.


8. dissident = dis~ (离开) + sid (坐) + ~ent (本质相关的)
             = (人们)分离的坐着
             = (adj.)持不同意见的


9. session = sess (坐) + ~ion (事物结果)
           = (人们)坐下来的状态
           = (n.)会期、授课期间

           「坐下来的时间」通常不是开会，就是上课。一般人以为这个字很冷门，其实不然，
           开会开得很久，可以说是a long session，倒没有人说a long meeting 的。
           另外，有科被当了，所以「暑修」，那叫做 summer session。
           可是这个字只是指「上课的日子」，不全然等于 semester 或 term。

10. assess = as~ (去使) + sess (坐)
           = 去使得(某事物能)坐定
           = (vt.)估价、课税


11. possess = pos~ (能够) + sess (坐)
            = (使)能够坐于(某事物之上)
            = (vt.)拥有、使沉迷

            pos~ 看起来是个很陌生的字首，其实不然，这个字首来自于 pot~，也就是 potenital、possible 的字根。


12. siege = assiege 的变体
          = as~ (去使) + siege (sid的变体，坐)
          = 对着(敌军)坐在那里 = (n.)围攻

          siege 这个字是从法文传进来的（之前那些字是直接从拉丁文
    进来的），字首的 as~ 也去掉了，方便发音的结果变成很难认得原字根字源。

13. assist = as~ (去使) + sist (坐)
           = 去坐在(某人身边)
           = (vt.)协助
           = help

           assist 是比 help 更正式的用字，因此是比较有规模的援助，小心有些搭配用语，
           如 come to one's assistance，不能代换为 help。


14. consist = con~ (一起) + sist (坐)
            = (人事物)聚集坐在一起
            = (vi.)存在、组合

            这个字的词性是不及物动词，因此后面介系词的搭配很重要。
            由于是「坐」的字根，因此这个字没有被动型态。
            consist of 凸显「本质」，所以是「构成」，
            consist with 凸显相互关系，所以是「与...一致」，
            consist in 凸显「位置」，因此是「在于...」。
            形容词 consistent 可能更接近字源的原意。

15. desist = de~ (向下) + sist (坐)
           = (某人)往下坐去
           = (vi.)放弃

           这个字是个很「象形」的字，无可奈何之时，往下一坐，表示放弃。
           这个字非常罕见，只有用在 desist from V~ing 这个句型之中。


16. exist = ex~ (在外) + sist (坐)
          = 一直坐在外头的(状态)
          = (vi.)存在、发生

          在这个字中，sist 不要一味强解为「坐」，「坐」其实很多时候只是凸显「存在」的意味，
          「存在在外面」所以「发生」

17. insist = in~ (往内) + sist (坐)
           = 一直坐在(某事物)里头
           = (v.)坚持、坚决要求

           同样地，这里未必是「坐」，只是表示一直坚持于某一点而不走

18. persist = per~ (完全) + sist (坐)
            = 始终坐在那里
            = (vi.)固执、坚持到底、持续

            由于字首per~ 的关系，这个字比 vinsist 更强调时间之长久，
            凸显「长时间地持续于某事」，所以慢慢也有负面意义产生，叫做「固执」。
            但是要小心这个字是中性的，是正面还是负面的意义，需要上下文表明。


19. resist = re~ (一再、向后) + sist (坐)
           = 一再地往后坐定
           = (vt.)抵抗、抑制

           resist 也是很「象形」的一个字：想像一下打电动玩具 VR 快打
           或是 Street Fighter 越打越往后坐的那种紧张姿态，不就是「抗御」的姿势吗？

20. subsist = sub~ (在...之下) + sist (坐)
            = 往下坐着(的状态)
            = (vi.)存活、维持生活、残留

            这个字变异很大，最早是「坐在事物之下」所以叫做「忍受」，慢慢变成「艰苦维生」
            跟 live 不同义，subsist 凸显「艰苦」，因此可以说 I live on my pension.，
            但只能说 I subsist only on bread and water. 凸显没有面包和水，我早就挂​​了。
            不过这个字很文言文，慢慢被取代中，预计几年内会消失。

【注】
~sed = ~sid 其实明眼人早看出来，这就是英文 sit 的前身，只是母音稍做改变而已。
~sess 不过是原形动词~sed 的过去分词形而已。而这个 ~sess 的字根跟 success 中那个 ~cess 发音纵然一样，实际意义却相差很多喔，
~sess 是「坐」，而~cess 是「走」。当然，同字根的 outside、inside、beside 那些简单的字在此就不做分析了，
只需要了解彼此相关而背单字时做联想即可。这个字根后来被英文借去变成 site，成为「地点」（坐着的地方），也造就出 situate 这个大家熟悉的动词。
可见字根字源一通，诸多单字之间的亲戚关系也瞬间恍然大悟，只是音节之间的转化而已~~
其实 ab~ 就是英文of的前身，be~ 就是英文 by 的前身，
而 ~sist 只不过比 ~sit、~sid 多个音节而已，这个 ~s~ 可能在历史的演变中逐渐因为发音考量而消失也未可知。
```
