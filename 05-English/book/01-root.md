基础字根

```
ab~ = from

con~ = com~ = 一起
de~ = 向下
dis~ = se~ = 离开
ante~ = pre~ = 在...之前
pro~ = 向前
per~ = 凭借、完全
en~ = em~ = in~ = im~ = 向內
re~ = 一再、向後
post~ = retro~ = 向后
ob~ = 朝向
sub~ = 在...之下
super~ = sur~ = 在...之上
inter~ = intro~ = 在...之間、互相
trans~ = meta~ = 橫过
anti~ = counter~ = 对抗 (against)
un~ = non~ = 否定
mis~ = 错误
bene~ = 好的
mal~ = 坏的
over~ = 过多
under~ = 不足
mono~ = uni~ = 一
bi~ = di~ = 二
```
