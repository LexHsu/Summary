字根九：~tain = ~ten = hold = 握持、拿
===

```
1. abstain = abs~ (ab~的变体，离开) + tain (拿)
           = 把(事物)拿​​离开
           = (vi.)节制、戒除

           abstain 几乎永远跟 from 连用，这个 from 也是从字首 ab~ 而来的，「拿离开」，
           所以 abstain from smoking 表示「戒烟」。
           很多用心的人会问：那跟 refrain 区别：
           refrain 只是一时的抑制（~frain 来自于「马缰」），
           abstain 常指长期的意志力控制
           repress 是抑制情绪。
           quit 和 get rid of 是指「戒除」的那个动作，
           abstain 是指长期的状态，所以下星期要戒烟，能用 quit，不能用 abstain。


2. contain = con~ (一起) + tain (握持)
           = 把(事物)握在一起
           = (vt.)包含、容纳
           = include

           contain 有个变体，叫做 content，~tent 就是 ~ten 的变体，
           因此 content 是 contain 的名词形，content 名词做「内容」可以理解，
           但是形容词做「满足的」这就打败很多人了，这个字原意是「充分包含所以满足」，
           该有的都有了。常跟学生开玩笑：con~ 叫做「一起」，tent 叫做「帐棚」，
           男女生露营通常一起躲在帐棚里，所以很满足。


3. continent = con~ (共同) + tin (ten的变体，握持) + ~ent (本质相关物)
             = 握持着所有(与民众相关)的事物
             = (n.)洲、大陆

             continent 是 contain 的变体字，~ent 本来是个形容词字尾，
             这里当名词用，这个字本来泛指所有民众聚集之大片土地，是蛮近代才演变为「一片大陆」的意思。

4. continue = con~ (一起) + tin (握持) + ~ue (拉丁文~um的变体)
            = 一起把(事物)握持住
            = (v.)持续


5. detain = de~ (向下) + tain (握持)
          = 把(人)往下拉
          = (vt.)耽搁、扣留

          detain 和 delay区别：前者只用于人，而后者大都用于事情。班机延误是后者而非前者。

6. entertain = enter~ (inter~ 的变体，在...之间) + tain (拿)
             = 在(人们)之间拿(食物)
             = (vt.)款待、使欢愉

             这个字起源于古早时，主人拿着食物宴请客人的场面

7. maintain = main~ (man~的变体，手) + tain (握持)
            = 用手握持住(事物) = (vt.)保持、维持
            = 手中握住(事物而发言) = (vt.)主张、坚称

  ★首先，man~ 这个字首并不罕见，manuscript、manual、manipulate...等
    字都跟「手」有关，会法文的同学自然知道法文中的「手」叫做main ~~
    这就是我常说的：学法文对于认识英文字汇有相当的帮助，~tain 这个字
    根在法文中是 tenir，很像吧？
    再者，maintain 当「主张」用，比较少人知道，但是不难领会：「手中抓
    着事物不放」自然就是坚称、主张的意象出来了。
    小心maintenance 这个名词通常指「机械的维修」以保持良好。

8. obtain = ob~ (朝向) + tain (拿) = 对着(事物)伸手拿
                                   = (vt.)获得 = gain = acquire
                                               = receive

9. attain = at~ (ad~的变体，去使) + tain (拿) = 去把(事物)拿​​过来
                                   = (vt.)获致、得到 = achieve

  ★attain 和obtain 这两个字也是学子们头痛的字辨，很麻烦的是从字源
    上看不出端倪（都是「去拿事物」），attain 着重于取得抽象之事物（如
    目标、结果那一类的字），而obtain 着重于取得实质上的物体（如薪
    水、奖赏那一类的字），所以我们会说：He attained the age of 85.
    而I obtained the first prize. 两者不能互换。

10. pertain = per~ (through) + tain (拿) = 穿过(某事物)而去拿目标物
                                         = (vi.)产生关连、附属

  ★我说过，per~ 是个很难的字首，原本是「穿过」而引伸为「完全」，你
    必须要穿过A 而拿到B，因此B 为A 的附属，而透过B 产生对A 的
    关连，这原本是个很简单的道理，只是per~ 慢慢意义增多，很多学子只
    记得「完全」的意义，那就失去学字根字源的本意了。这也是为什么有时
    候「原文书」比翻译书管用的原因，因为一个through 表达的观念，在中
    文中必须转好几个圈子。
    至于pertain 后面为啥接to，我想这已经是很简单的问题了：表达「两者
    之间相对、附属的关系」不用to，不然呢？

11. retain = re~ (向后、回来) + tain (握持) = 把(事物)拿​​回来
                                            = (vt.)保留、维持、记得

  ★很多人一看到「维持」就跟maintain 搞不清了，请记得re~ 是「回来」
    的意味，因此retain 强调「保持不希望失去的事物」，而maintain 用
    手握持住，只是「期待保持原状」而已，因此retain my dignity、retain
    all the past events，而 maintain world peace。
    而这里的retain 跟reserve 也不同，后者是「事先留存」以备后用。

12. sustain = sus~ (sub~的变体，在下方) + tain (握持)
            = 在下方拿着(事物) = (vt.)支撑、承受、供养、证实

  ★之所以会有sus~ 这种变异字首跑出来，是因为一开始是subtain，因为
    难念，所以加上s，变成substain，念快了、念久了，那个b 消失掉了，
    变成sustain，事实上，就这三者而论，的确sustain 是最容易发音的结
    果。
    由于字源是「在下头握持事物」，因此这个字跟maintain 不同，这个字着
    重于「努力去持续、忍受」，所以sustain a big family、sustain a
    terrible shock、sustain injuries 这些都是maintain 所无法取代的。
    sustain 最后一个意义「证实」，也不难，「有事实在下头支撑」，所以
    叫做「证实、赞成」，这个字经常在法庭可以听得到，当有律师大喊：
    Objection！的时候，法官不是讲 Sustained！就是讲 Overruled！

13. tenacious = ten (握持) + ~ac (~ate的变体，使得) + ~ious (多...的)
              = 一直握持着的(状态) = (adj.)不屈不挠的、固执的

  ★很多人会问：好好的~ate 或~at 为啥变成~ac，很好的问题。英文字汇有
    七成以上传承自拉丁文没错，但不要忘了法文才是中间传承的媒介（法国刚好
    在义大利和英国的中间点），~ate 的拉丁形式在法文中软化为~ac，方便发
    音，也比较不混淆视听。事实上，法文几乎软化了六成以上拉丁的子音，在
    拉丁文是硬音（如[g]、[b]...等）进到法文都软化了（如[d3]、[v]...等）
    但这是题外话了，我大四那一年修『语概』时，曾经写过十四页的期末报告
    讨论法文把拉丁文子音软化的情况和整理。当然，法文把子音软化的结果，
    就是造成大家都认为「法文最优美、好听」的误解来源之一。在此不多提，有
    兴趣的网友请自行上140.112.34.34 站上有个版叫做Vernie，其中有收录这
    篇期末报告的精华本。

14. tenant = ten (握持) + ~ant (接受动作者) = 被握持住的人
                                            = (n.)房客、租屋者、佃农

  ★~ant 或~ent 本来都是形容词字尾，表示「与...本质相关的」，转变为名
    词之后，纯粹表示「与动作本质相关之人事物」，所以在这里说「接受动作
    者」其实是不精准的，只是方便解析而已，正确该解为「与握持这动作相关
    之人或物」，虽然也可以导出最终意义，只是不通顺罢了。

15. tenet = ten (握持) + ~et (小事物) = 手中握持的小事物
                                      = (n.)信条、教义

  ★~et 和~ette 都是「小」的意思，如cigar 和cigarette、pig 和piglet
    一般。要特别把这个字尾拿出来讲是有意义的，因为tenet 和principle、
    dogma、creed 不一样，tenet 特指「个人或党派信奉之主义」，外界未必
    认同，所以是「手中握的小东西」，而principle 指「获得证实之原理」、
    dogma 指「教会制订之教义」、creed 指「教义之概要」，由此可分出字义
    上使用之差异，不可不慎！

16. tenure = ten (握持) + ~ure (事物状态) = 握持有之事物状态
                                          = (n.)保有权、任期

【后记】~tain、~ten 这个字根其实并不难解，而我之所以把它留到这么后面的
        原因是因为这一系列的字都很难，不只是难用，而且是字辨一堆，容易
        混淆的字一大串，所以读者可以看出这一篇之中，讲解之多可能胜过过
        去每一篇，这一串字难不在于解字、而在于厘清，要清晰解释出字与字
        之间的差异和用法比背单字还难，这就是为什么背单字必须经历「字辨
        法」的折磨~~ 英文的近义字之多，全世界之冠（也因为其单字量太多）
        ，所以读字典务必小心而思考灵活，死读字典的人也许能获得高分，但
        缺乏自主思考能力，终难列入高手之林。一面读字典，一面做整理（在
        笔记本中或是在脑中）才是高明，背单字务求前后连贯、旧字与新字产生
        关连，这样才能克服难关。所以我说：我不求经由这套方法各位能够多
        背好多单字，只求各位学习的态度能够改观，仅此。
```
