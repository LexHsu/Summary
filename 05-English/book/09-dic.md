字根八：~dic = ~dict = ~loqu = ~logue = say, speak = 说话
===

```
1. diction = dict~ (说话) + ~ion (动作状态)
           = 说话(时用)的动作
           = (n.)发音法、措辞

           dictionary 来自于该字根，把发音法和措辞记录下来的事物称为字典，不过份。


2. dictate = dict~ (说话) + ~ate (使得)
           = 使得(某人)说话
           = (vt.)强硬要求 (vi.)口述

           dictate 是强硬要求使不开口的人说话，如果不行，衍生为帮他做「口述」，
           dictator 是去使别人开口的人，衍生为「专横的独裁者」。
           而 dictation(口授，听写；听写测验) 可联想上语言课时，老师念、要求我们听写，一种听写测验手段。


3. predict = pre~ (在之前) + dict (说话)
           = 在(事情发生)之前说出
           = (vt.)预言
           = foretell
           = prophesy


4. indict = in~ (往内) + dict (说话)
          = 去说某人(隐私)的事
          = (vi.)起诉、控告

          indict 的发音非常奇怪，是十八世纪英国文人追求风雅，模仿拉丁发音搞出来的不伦不类的发音。
          再者，indict 原本是「把人家的隐私、罪恶翻出来」的意思，在现代，这样搞叫做「毁谤」，
          因此演变成在法庭上掀人私事的行为，不过这个字很文言文，一般都用 charge。


5. edict = e~ (ex~的变体，向外) + dict (说话)
         = 把(事件)对外说出
         = (n.)告示、命令
         = order
         = command

         edict 这个字 ex~ 简化为 e~ 是考量到发音原则，这个字很文言文，知道即可。


6. indicate = in~ (向内) + dic (说话) + ~ate (使得)
            = 外在的(人事物)去描述内在的(人事物)
            = (vt.)指出、指示、显示有...需要

            对旁边的人说明某事物内在的状况，自然就是「指出」。


7. abdicate = ab~ (from，离开) + dic (说话) + ~ate (使得)
            = (大家)说话使得(某人)离开(职位)
            = (vt.)放弃(职位、权利)、退位、让位


8. dedicate = de~ (向下) + dic (说话) + ~ate (使得)
            = (人)一直朝着下头(人群)说话
            = (vt.)致力于、奉献
            = devote

            这个字起源于古罗马的演讲者，如 Cicero 之流的人物，对着群众奉献自己的心力和智慧，
            一篇又一篇精彩的说词奠定了罗马帝国的人文基础，这个字用法 dedicate A TO B。


9. addict = ad~ (去使) + dict (说话)
          = 去对着(人事物一直)说话
          = (vt.)使耽溺于、使沉迷

          这个字是从情爱方面演变而来，一个男孩子一天到晚对着自己心爱的女孩说话，自然是「耽溺、沉迷」！
          但是留意，这个原形动词几乎不用，九成九以上是：They are addicted to television.
          addict 也可做名词，变成「耽溺沉迷者」，xx adict = 沉迷于 xxx 者。
          如 Japanese drama addict「日剧沉迷者」


10. benediction = bene~ (好) + dict (说话) + ~ion (状态)
                = 说好话的事
                = (n.)祝福

                bene~ 这个字首就是法文传进来的，它的相对词是 mal~（恶劣的），


11. contradict = contra~ (相反) + dict (说话)
               = 说相反的话
               = (vt.)反驳、抵触

               contradict 和 refute 的差别：
               contradict 后面通常是接「人」，表示「顶撞」，如果接「事」，表示「互相抵触、互相矛盾」；
               refute 则是「驳斥」某想法或理论
               Our opinions contradict with each other 是画蛇添足，
               Our opinions contradict each other 即可，无须介系词。
               就算要填介系词，再怎样也轮不到with，看到 contra~ 这字首就知道两者是相对关系，要用 to：
               Your behavior is in contradiction TO your principles.


12. interdict = inter~ (between，在其间) + dict (说话)
              = 话语介入人们之间 = (n.)(宗教)禁令

              这个字超级文言，可能一辈子也用不到一次，一般社会上的「禁令」我们口语会用 ban


13. verdict = ver~ (真实) + dict (说话)
            = 说出真实的话
            = (n.)法庭判决、裁定

            ver~ 这个字首很少见，来自于拉丁文的 verum，就是 true 的意思，
            也就是英文 very 的前身，very 这个字在中古英文是 really 的意思。
            古老的人文精神期望着执法机关做出最后而真实的结论。


14. jurisdiction = juris~ (法律) + cict (说话) + ~ion (动作状态)
                 = 法律表达给(人民)的状态= (n.)司法权、管辖权

                 juris 来自于拉丁文的 jus，就是英文 just 的前身，
                 所以 just 有「公正」的意思在其中。这个字也很正式。


15. condition = con~ (一起) + dit (dict的变体，说话) + ~ion (动作状态)
              = 大家所描述的状态
              = (n.)状态、条件(vt.)使适应于

              condition 泛指抽象的存在状况，「汽车保养状况」用condition。
              situation 着重于某地方当时的四周情况，「政治现状」用situation

              condition 当动词用，护发乳叫 hair~conditioner，空调叫做 air~conditioning。


16. edit = e~ (向外) + dit (dict的简化，说话)
         = 使得(事物)​​能够对外说出去
         = (vt.)编辑


17. index = in~ (往内) + dex (dict的变体，说话)
          = 说明(事物)内容之物
          = (n.)索引、指标


18. colloquial = col~ (com~的变体，一起) + loqui (说话) + ~al (相关的)
               = 与大家说话所相关的
               = (adj.)口语的、会话的

               colloquial 和 idiomatic 是不一样的字，不能代换！
               idiom 是表示惯用语，所以 idiomatic 是表示惯用语可以接受的，或是很地道的。
               而 colloquial 只是表示「谈话中会用到的」，谈话中用到的词句未必是惯用语、也未必地道。


19. eloquent = e~ (ex~的变体，往外) + loqu (说话) + ~ent (本质的)
             = (某人)具有对外说话的本质的
             = (adj.)雄辩有口才的

  ★像是希特勒或是陈水扁在演讲，都可以说他们是eloquent 的人，但可别
    说他们是 talkative 喔！因为后者是「多嘴、饶舌的」的意思。还要小心
    一点：eloquent 有个近亲，叫做elocution，loqu 变成locu，故意染头
    发装成ABC，但是骨子里那个音一听还是原形毕露。

20. soliloquy = soli (solo的变体，单一) + loqu (说话) + ~y (动作结果)
              = 一个人说话的状态结果
              = (n.)独白、自言自语

21. circumlocution = circum~ (环绕) + locu (loqu的变体，说话) + ~tion (动作状态)
                   = 说话绕圈子的状态= (n.)婉转曲折的说法、迂回

                   这个字就是英文 circle 的始祖。circumlocution 是给那些说话言词闪烁、闪闪躲躲的人，
                   直接给他们一句：Stop all that circumlocution ~~ just give me the truth!


22. prologue = pro~ (在前) + logue (说话)
             = 在前头叙述(事物)的事物
             = (n.)序曲、开端


23. epilogue = epi~ (在其上) + logue (说话)
             = 在(事物)之上描述(全部)
             = (n.)后记、跋

             有prologue、有epilogue，是否有前有后就该有个中央，叫做interlogue？
             看似合理，可是抱歉没这个字，「插曲」叫做 interlude。


24. catalogue = cata~ (向下) + logue (说话)
              = 叙述其下之事物
              = (n.)目录 (vt.)编入目录

              ata~ 是个希腊字首，相当于拉丁的de~，不常出现。


25. monologue = mono~ (单一) + logue (说话)
              = 某人独自说话
              = (n.)长篇独白
              monologue 与 soliloquy 这两个字的区分无法从字源上得到解答（因为字源完全一样）：
              soliloquy 是某个主角对着观众自言自语，诉说自己心中感受，沉思者有开口说话；
              monologue 则是作者或是编剧帮沉思者加上的心灵注解，沉思者没有开口说话。


26. dialogue = dia~ (through，穿过) + logue (说话)
             = 穿过(人与人)的话语
             = (n.)对话、对白

             希腊文的 dia~ 就是拉丁文的 per~。dialogue 跟 conversation 区别：
             conversation 可能只在口头上进行，可能印刷在教科书上，
             dialogue 只能表示「记录下来的对话」。
             所以书上的会话课程是 conversation，也是 dialogue，
             但是与朋友谈了一会儿话很开心，叫做 conversation，而不是 dialogue。


27. travelogue = travel (旅行) + logue (说话)
               = 谈论旅游之事
               = (n.)游记、旅行见闻

               注意虽然有两个 l，可是结合之后只剩下一个 l，这是「字根」的特色，
               会融合、会简化，跟单字不同，比如说 roommate 两个字结合却不能简化为roomate，
               因为字尾 ~ate 会让人联想到「使得」那个动词字尾，造成干扰。


28. apology = apo~ (离开) + log (logue的变体，说话) + ~y (结果)
            = (要)离开得说的话 = (n.)道歉

            apo~ 是阿希腊字根，与跟拉丁文的 ab~ 同源（古印欧语），（再如 phor 和 port 和 fer）
            英文中有个漂亮句型： A is a sad apology for B


29. eulogy = eu~ (好) + log (说话) + ~y (结果)
           = 说好话的事
           = (n.)歌功颂德

           eu~ 就是拉丁文的 bene~，都是「好」的意思，
           例如大家熟知的「安乐死」就是euthanasia，其字源就是「死得好」。
           要记住：benediction 通常是好的「祝福」，而 eulogy 通常是不好的「奉承讨好」。
```
